# Linux Homelab Infrastructure

Este repositório documenta meu laboratório pessoal de infraestrutura Linux, criado para estudo prático de virtualização, containers, redes e administração de servidores.

O ambiente simula uma pequena infraestrutura corporativa utilizando tecnologias amplamente utilizadas em ambientes profissionais.

## Tecnologias utilizadas

- Linux
- KVM / Libvirt
- Docker
- WireGuard VPN
- Nginx
- Shell Script
- Networking

## Arquitetura do ambiente

Servidor físico:

Dell OptiPlex 3020  
Intel i5  
8 GB RAM

Host:

Linux (Zorin OS)

Virtualização:

KVM + Libvirt

Containers:

Docker

VPN:

WireGuard

## Serviços implantados

- Nginx container
- Portainer
- VPN WireGuard
- Máquinas virtuais Linux para laboratório

## Objetivos do laboratório

- Estudar infraestrutura de servidores
- Praticar virtualização com KVM
- Testar containers Docker
- Simular ambientes corporativos
- Evoluir conhecimentos para áreas como infraestrutura e SAP Basis

## Estrutura do repositório

architecture → diagramas da infraestrutura  
virtualization → configuração de máquinas virtuais  
docker → serviços em containers  
vpn → configuração da VPN  
networking → topologia de rede  
scripts → automações administrativas
